# Master Math Challenger

A responsive web app where families and friends compete on multiplication speed and accuracy.

## Features

- Email/password sign up and login
- User profiles (display name + optional age/grade)
- Group creation, invite-code joining, and member lists
- Competition creation (group or ad-hoc), invite-only/public group visibility
- Competition leaderboard ranked by total correct answers with average time tie-breaker
- Adaptive multiplication sessions (30 questions, level 1-10)
- Session logging per question and session summary metrics
- Dashboard with daily totals (time, sessions, questions, accuracy)
- Personal stats with best-session cards, history table, and accuracy trend chart

## Run locally

```bash
npm install
npm run dev
```

## Build and lint

```bash
npm run lint
npm run build
```
